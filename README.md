# Containerized Node.js Application using AWS ECS & ECR

## Project Overview

This project demonstrates containerizing a Node.js application using Docker and deploying it on AWS ECS Fargate with Amazon ECR.

## Architecture

Node.js Application
↓
Docker Container
↓
Amazon ECR
↓
Amazon ECS (Fargate)
↓
Running Web Application

## Technologies Used

* Node.js
* Express.js
* Docker
* Amazon ECR
* Amazon ECS
* AWS Fargate

## Features

* Dockerized Node.js application
* Image storage in Amazon ECR
* Serverless container deployment using ECS Fargate
* Scalable cloud-native architecture

## Deployment Steps

1. Create Node.js Application
2. Build Docker Image
3. Push Image to Amazon ECR
4. Create ECS Cluster
5. Create Task Definition
6. Deploy Service on ECS Fargate
7. Access Application via Public IP

## Learning Outcomes

* Containerization with Docker
* AWS Container Services
* ECR Image Management
* ECS Task Definitions
* Fargate Deployment
* Cloud Application Hosting
